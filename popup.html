Texte source

Ajouter un champ

Nom du champ tel qu'il apparaîtra dans le résumé (ex: "Pression Min"). Utilisez des noms simples (CamelCase possible). Uniques par prestataire.
Choisissez les positions à garder. Pour taille = 1, c'est toujours "X".
📋 Mot clé vide (lignes 1-999) ▲
Si la ligne du label contient un de ces mots, cette occurrence est ignorée (utile quand un même mot-clé apparaît en plusieurs contextes).
Les lignes de valeur contenant un de ces mots sont ignorées. Utile pour filtrer des lignes parasites juste après le label.
Parmi plusieurs valeurs candidates, préfère celle qui contient un de ces mots. Laissez vide pour prendre la première valeur trouvée.
Laissez vide si vous ne souhaitez pas ajouter de mot clé pour l'instant.
Définit comment ce champ sera interprété (seuils actuels: Observance ≥ 4h, IAH ≥ 5/h, Fuites ≥ 24 L/min).
